Day 8 - MongoDB (Database Layer of Full Stack)

1. Introduction to MongoDB in Full Stack

  • Role of MongoDB in MERN stack.
  • Stores application data in JSON-like documents.
  • Works seamlessly with Node.js (backend).
  • Provides flexibility (no strict schema like SQL).

2. Setting Up MongoDB

  • Install MongoDB & MongoDB Compass.
  • Cloud Alternative: MongoDB Atlas (free tier).
  • Create first database (fullstackDB) and collection (users).
